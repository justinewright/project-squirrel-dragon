<h2 align="center">Collect. Track. Trade.</h2>

## Project Description
Squirrel Dragon is an iOS application. It is developed for pokemon card collectors to help them keep track of their sets and value.

### Built With
* Swift 
* Obj-c
* MVVM
* Rest-api
* WatchKit
* Firebase
